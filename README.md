# Customer_behaviour-_analysis
Built an end-to-end customer shopping behavior analysis project using Python, PostgreSQL, SQL, and Power BI. Performed data cleaning, SQL analysis, and created an interactive dashboard to analyze customer purchasing patterns and business insights.

Overview:

This project focuses on analyzing customer shopping behavior using transactional retail data. The objective was to identify purchasing trends, customer segments, category performance, and subscription behavior through an end-to-end analytics workflow.

The project demonstrates the complete data analytics pipeline:

Data Cleaning & Feature Engineering using Python
Database Integration with PostgreSQL
Business Analysis using SQL
Interactive Dashboard Development in Power BI


Project Architecture:
CSV Dataset → Python (EDA & Cleaning) → PostgreSQL Database → SQL Analysis → Power BI Dashboard.
